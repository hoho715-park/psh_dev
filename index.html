About Me

유연하게 소통하고 견고하게 개발합니다

사용자 중심의 프런트엔드 설계

React를 활용하여 재사용 가능한 컴포넌트 기반의 UI를 구축하고, 사용자 경험(UX)을 극대화하는 반응형 웹을 구현합니다. 상태 관리 라이브러리와 최신 훅(Hooks)을 사용하여 복잡한 비즈니스 로직을 효율적으로 처리합니다.

안정적인 백엔드 및 API 구축

Spring Boot와 JPA를 사용하여 대규모 데이터를 안정적으로 처리하는 REST API 서버를 설계합니다. 계층화된 아키텍처를 준수하여 유지보수가 용이한 코드를 작성하며, 보안과 확장성을 고려한 백엔드 환경을 구축합니다.

효율적인 협업을 위한 툴체인 활용

GitHub 기반 Git-flow 전략을 활용한 체계적인 코드 관리와 코드 리뷰를 실천합니다. Notion으로 요구사항과 컨벤션을 문서화하고, Figma를 통해 디자인 의도를 정확히 파악하여 구현에 반영합니다.
